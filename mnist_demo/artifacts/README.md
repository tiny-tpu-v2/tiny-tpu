# Artifacts

This folder collects generated outputs from the MNIST demo so Quartus, simulation, preview, benchmark, and runtime files stay out of the source directories.

- `benchmarks/`: outputs from benchmarking utilities.
- `previews/`: generated image and HTML previews.
- `quartus/`: archived Quartus build outputs for the FPGA revisions.
- `runtime/`: host-side captures and logs from live demo runs.
- `sim/`: ModelSim transcripts and compiled work libraries.
